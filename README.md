# **💻 How to Type These on Linux Mint (Compose Key)**

Enable:

**Keyboard Settings → Layout → Options → Compose Key → choose Right Alt**

Then:

-   ➜ = Compose + `-` + `>`
-   ➤ = Compose + `>` + `>`
-   — = Compose + `-` + `-` + `-`
-   … = Compose + `.` + `.` + `.`
